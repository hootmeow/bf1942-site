# Map Images Guide

Each map folder should contain the following images:

## Required Images Per Map

### Hero Image
- `hero.jpg` - Main overview/cinematic shot of the map (1920x1080 recommended)

### Gallery Images
- `gallery-1.jpg` - Gameplay screenshot 1
- `gallery-2.jpg` - Gameplay screenshot 2
- `gallery-3.jpg` - Gameplay screenshot 3
- `gallery-4.jpg` - Gameplay screenshot 4

### Control Point Images
For each control point, add an image named after the point:
- `cp-a.jpg` - Control Point A screenshot
- `cp-b.jpg` - Control Point B screenshot
- `cp-c.jpg` - Control Point C screenshot
- etc.

## Image Specifications
- Format: JPG or PNG
- Recommended size: 1280x720 minimum
- Aspect ratio: 16:9 preferred
